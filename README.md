# TwitViz

## Description

### Currently

A simple smooth-scrolling display for tweets for specific hash tags (pycon
as configured)

### Future

A set of visualizations that will display various interesting data about
the current state of the twitterverse.

## Requirements

* Redis
* PyYaml
* PyProcessing
** Pyglet
** OpenGL

## To Run

* Start redis-server
* Start streamer.py
* Start viz.py
